# Voyra
Voyra: The Smart Explorer  Voyra is an enterprise-grade travel intelligence platform engineered to orchestrate corporate travel and off-sites across Asia. Built as a professional, service-oriented agent, it bypasses generic travel aggregators to deliver highly grounded itineraries.By leveraging Microsoft's intelligence layers, Voyra ensures that travel schedules are cost-effective, logistically sound, and strictly aligned with organizational constraints and verified local data.
Key Features

Context-Aware Scheduling: Automatically extracts travel parameters, team availability, and trip objectives directly from internal communication channels.
Authentic Local Routing: Discards generic itineraries in favor of verified, localized insights to construct practical and efficient travel routes.
Enterprise Financial Gatekeeping: Cross-references proposed itineraries against live corporate expense policies to enforce strict budget compliance.
Traveler Utility Suite: Includes practical, built-in tools such as automated baggage weight calculators and offline-ready itinerary storage via local browser persistence.
Continuous Intelligence Loop: At the conclusion of a trip, the system automatically queries the traveler regarding their experience, integrating this feedback into the primary repository to ensure all local intelligence remains current and accurate.

Architecture & Technology Stack

The platform is developed independently, utilizing a highly modular architecture to separate the user interface from the enterprise AI layers without relying on external cloud hosting infrastructure like Azure.
Client Interface: A clean, responsive dashboard built with semantic HTML, CSS, and vanilla JavaScript. It is designed purely for service utility and clear data presentation, avoiding unnecessary visual clutter.
Backend Services: A secure, custom-built backend utilizing object-oriented Java principles to handle data routing, application logic, and secure synchronization.
Work IQ: Integrates with the Microsoft Graph API to parse authorized enterprise calendar events and collaboration data for context.
Fabric IQ: Connects to organizational semantic models to analyze and compute live travel budgets against proposed costs.
Foundry IQ: Acts as the standalone retrieval layer, grounding the agent in a secure, locally managed knowledge base of Asian travel logistics and corporate policies to ensure hallucination-free routing.
